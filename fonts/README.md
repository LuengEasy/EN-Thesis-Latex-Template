# Fonts

This template does not bundle commercial or system fonts.

By default, `main.tex` looks for Century in this order:

1. `fonts/CENTURY.TTF`
2. `C:/Windows/Fonts/CENTURY.TTF`
3. `TeX Gyre Schola`

If you need the exact Century font, copy or upload the font file yourself:

```text
fonts/CENTURY.TTF
```

On Overleaf, the file name is case-sensitive. Use `CENTURY.TTF` exactly.
